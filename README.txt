# 高健的个人主页项目 (Personal Website Project)

这是一个简洁、美观且响应式的个人主页，专为学术展示设计。

## 目录结构
- `index.html`: 网页主文件，双击即可在浏览器查看。
- `a.txt`: 原始论文列表数据。
- `refresh_publications.py`: (可选) 当你更新 `a.txt` 后，运行此脚本可以自动同步到网页。
- `avatar.jpg`: (需准备) 请将你的个人照片命名为 `avatar.jpg` 放在此目录下。

## 如何使用
1. **添加照片**: 准备一张个人照片，重命名为 `avatar.jpg` 放入文件夹。
2. **查看网页**: 直接双击 `index.html`。
3. **移动文件夹**: 整个文件夹可以移动到任何地方，网页依然可以正常运行。
4. **更新论文**: 
   - 修改 `a.txt` 中的内容。
   - 如果你会使用 Python，运行 `python refresh_publications.py` 即可更新网页（目前网页已包含全部23篇论文）。

## 网页功能
- **自适应**: 支持手机、平板和电脑查看。
- **美观大方**: 基于 Tailwind CSS 设计，风格简洁专业。
- **交互**: 论文列表支持折叠显示，点击导航栏可平滑滚动。
