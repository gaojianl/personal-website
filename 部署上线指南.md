# 如何将您的个人主页上线 (GitHub Pages 部署指南)

要让其他人通过互联网访问您的个人主页，最简单、免费且专业的方法是使用 **GitHub Pages**。

### 第一步：准备工作
1. 注册一个 [GitHub](https://github.com/) 账号（如果还没有）。
2. 下载并安装 [Git](https://git-scm.com/)。

### 第二步：创建仓库
1. 登录 GitHub，点击右上角的 **+** -> **New repository**。
2. 仓库名（Repository name）建议填：`personal-website`。
3. 选择 **Public**。
4. 点击 **Create repository**。

### 第三步：上传文件
在您的电脑文件夹 `c:\Users\gj\Desktop\PersonalWeb` 中，右键选择 "Open Git Bash here"（或在终端中进入该目录），依次运行以下命令：

```bash
# 初始化
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial personal website"

# 关联远程仓库 (将 <your-username> 替换为您自己的 GitHub 用户名)
git remote add origin https://github.com/<your-username>/personal-website.git

# 推送代码
git branch -M main
git push -u origin main
```

### 第四步：开启部署
1. 在 GitHub 仓库页面，点击顶部的 **Settings**。
2. 在左侧菜单点击 **Pages**。
3. 在 **Build and deployment** 下的 **Branch** 部分，确保选择了 `main` 分支和 `/ (root)` 文件夹。
4. 点击 **Save**。

### 第五步：完成上线！
等待 1-2 分钟，GitHub 会给您一个链接（通常是 `https://<your-username>.github.io/personal-website/`）。现在任何人都可以通过这个链接访问您的个人主页了！

---
**提示**：
- 如果您以后更新了 `index.html` 或图片，只需再次运行 `git add .`、`git commit -m "update"` 和 `git push` 即可。
- 校徽和个人照片必须随 `index.html` 一起上传。
